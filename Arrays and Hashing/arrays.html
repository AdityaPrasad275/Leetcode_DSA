Jist of arrays

  • Sorting: An $O(n \cdot logn)$ process. Quick sort, merge sort, Bubble sort ($O(n^2)$)
    • std::sort(vector.begin(), vector.end()) => can be considered $O(n)$ for all practical purposes
  • For string problems, maintaining frequency maps is often required. One can use an array or hashtable for it. In general, std::unordered_map is very useful in "grouping" problems
    • std::array<int, 26> freq;
    • std::unordered_map<char (or int), int> freqMap;
  • std::unordered_set<int> is a great data structure. $O(1)$ insertion, $O(1)$ search and no duplicates (for std::set, everything is $O(logn)$)
